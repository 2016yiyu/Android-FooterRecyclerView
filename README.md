# Android-FooterRecyclerView
通过自定义RecyclerView实现上拉加载功能

###使用配置<br>
_ _ _ _
####1.Add root build.gradle<br>
```
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
#####2.build.gradle
```
dependencies {
	        compile 'com.github.2016yiyu:Android-FooterRecyclerView:v1.0'
	}
```
